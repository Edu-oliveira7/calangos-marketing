## 📖 Sobre o Projeto

A **Calangos Marketing** é uma landing page institucional criada para apresentar a agência, seus serviços e facilitar o contato com potenciais clientes.

O projeto foi desenvolvido com foco em **performance, design moderno e responsividade**, garantindo uma experiência fluida em dispositivos desktop e mobile.

---

## ✨ Funcionalidades

| Recurso | Descrição |
|---|---|
| 📱 Responsivo | Layout adaptado para todos os dispositivos |
| 🎨 Design moderno | Interface limpa com identidade visual própria |
| ⚡ Alta performance | Build otimizado com Vite |
| ✨ Animações | Transições suaves com Framer Motion |
| 📩 CTA | Seção de chamada para ação estratégica |
| 📷 Instagram | Integração com redes sociais |
| 📧 Contato | Formulário de contato direto |
| 🚀 SEO | Otimização básica para motores de busca |

## 🛠 Tecnologias

| Tecnologia | Uso |
|---|---|
| [React](https://reactjs.org) | Biblioteca principal de UI |
| [Vite](https://vitejs.dev) | Bundler e servidor de desenvolvimento |
| [Tailwind CSS](https://tailwindcss.com) | Estilização utilitária |
| [Framer Motion](https://www.framer.com/motion) | Animações e transições |
| [Lucide React](https://lucide.dev) | Ícones |
